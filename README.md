# Retail Sales and Profitability Analysis for Business Decision-Making

## Project Overview
This project analyzes retail sales data to understand performance across categories, regions, states, customer segments, discounts, and profitability.

The goal of this analysis is to identify high-performing areas, detect loss-making segments, and generate business recommendations based on sales and profit patterns.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook
- VS Code

## Analysis Performed
- Data cleaning and preprocessing
- KPI calculation
- Category-wise sales and profit analysis
- Region-wise and state-wise profitability analysis
- Segment performance analysis
- Discount vs profit analysis
- Sub-category loss analysis

## Key Findings
- High sales did not always lead to high profit.
- Some sub-categories showed low or negative profitability.
- Discount levels appeared to affect profit margins.
- Regional and state-level performance varied significantly.
- Customer segments contributed differently to sales and profit.

## Business Recommendations
- Reassess discounts on low-profit products.
- Focus on high-profit categories and markets.
- Investigate underperforming states and product groups.
- Use profit alongside sales for strategic decisions.
